経費精算・報告システム (DX版)

三菱重工コンプレッサ株式会社における生産技術・DX推進業務の効率化、および属人性を排除した「止まらない運用」を志向した経費精算ツールです。

🛠 コンセプト

RACIの明確化: 担当者による送信アクションをデジタルログとして確定。

トレーサビリティ: 支出の目的、内容、金額をスプレッドシートへ自動蓄積し、後日の監査や解析（σ箱等）に接続。

モバイルファースト: 現場での入力を容易にする「大人かわいい」かつ「直感的な」UIデザイン。

🚀 機能

Google Apps Script (GAS) 経由でのGoogleスプレッドシート自動連携。

Gemini AI によるモチベーションアップ用マスコットの自動生成。

印刷・PDF出力への最適化（監査対応用）。

📈 今後の展望

数理統計（ANOVA、GMM等）を用いた予実管理の自動化。

広島市・三菱重工コンプレッサのデジタルツイン基盤へのデータ統合。
